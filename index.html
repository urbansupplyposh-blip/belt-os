What is Belt OS?

Belt OS is a multi-agent operating system for your business. It's a carbon copy of Muddy OS, rebuilt for the Belt team with all the same features and visual design.

Architecture

Belt OS runs on a hub-and-spoke model. Waul (Manager) coordinates everything, while Bobby, Maria, and Tim handle specialized tasks.

Belt OS is built with pure HTML/CSS/JavaScript - no frameworks, no build process. It deploys directly to GitHub Pages and works on any device.
